# NetStats v1.0.0

NetStats is a native macOS menu bar monitor for CPU, memory, network speed, public IPv4, geolocation, and Clash Verge Dev status.

## Install

### Homebrew

```bash
brew tap autumncry/tap
brew install --cask netstats
```

### DMG

Download `NetStats-1.0.0.dmg`, open it, then drag `NetStats.app` to `/Applications`.

This first public build is unsigned and not notarized. macOS may block the first launch. Open **System Settings > Privacy & Security** and allow NetStats, or Control-click the app and choose **Open**.

## Highlights

- Compact macOS menu bar display for CPU, memory, and network speed.
- Detail panel with hardware, memory, network, public IP, geolocation, and Clash Verge Dev status.
- Public IPv4 copy button.
- Chinese and English UI.
- Configurable menu bar and hover metrics.
- MIT licensed source code.

## Privacy

System metrics, network speed, and Clash Verge Dev status are read and processed locally on your Mac and are never sent to any NetStats-owned server. Public IP and geolocation lookup uses `ipinfo.io` to resolve the current public egress address.
