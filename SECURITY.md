# Security Policy

## Supported Versions

NetStats is early-stage software. Security fixes are applied to the latest release and the `main` branch.

## Reporting a Vulnerability

Please report security or privacy issues by opening a GitHub issue with minimal reproduction details, or by contacting the repository owner through GitHub.

Avoid posting private IP addresses, proxy configuration, node names, or screenshots containing personal desktop information. If a public issue would expose sensitive information, share only the affected feature and a high-level description first.

## Privacy Expectations

NetStats reads system metrics, network speed, and Clash Verge Dev status locally. Public IP and geolocation lookup uses `ipinfo.io` to resolve the current public egress address. Any future network behavior should be documented before release.
